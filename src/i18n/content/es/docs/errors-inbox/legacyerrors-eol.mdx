## Actualizaciones de agente requeridas

Ciertas versiones anteriores del agente no producen `TransactionError` evento, que son necesarios para la nueva experiencia. Para garantizar la funcionalidad completa, actualice a las siguientes versiones mínimas del agente o superiores:

* Ruby: 3.14.0.305
* PHP: 5.1.130
* .NET: 6.8.172.0
* Java: 3.21.0
* Python: 2.58.0.43
* Nodo.js: 1.24.0
* Go: 0.5.0

## Actualice para la nueva experiencia de errores

Recomendamos encarecidamente actualizar a las últimas versiones del agente para aprovechar al máximo la nueva experiencia de errores.

Para obtener instrucciones detalladas sobre cómo actualizar agente, consulte [nuestra documentación](/docs/new-relic-solutions/new-relic-one/install-configure/update-new-relic-agent/).
