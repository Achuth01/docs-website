Sabemos que su código fuente es uno de sus activos más valiosos y sensibles. Para la mayoría de los equipos de desarrollo, es una representación literal de su producto de trabajo más importante.

CodeStream necesita almacenar fragmentos de código en nuestra base de datos cuando discutes un bloque de código a través de un comentario o problema. CodeStream guarda ese bloque de código en la base de datos por diversas razones. Por ejemplo, si el código en cuestión no ha sido enviado, almacenarlo es la única forma de compartirlo con tus compañeros de equipo.
